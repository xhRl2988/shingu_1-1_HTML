tag정리(김현수)

  • h1~h6: 크기조절
  • html: 최상위 태그 html를 시작하는 태그
  • head: 타이틀 적는곳
  • body: 화면에 보이는 곳
  • ol: 순서있는 항목의 목록을 나타내는 태그
  • ul: ol태그에서 순서만 없는 태그
  • String: 볼드
  • b: 볼드
  • i: 기울이기
  • em: 기울이기
  • sup: 주석같은거
  • sub: 주석같은거
  • img: 이미지를 넣는 태그
  • u: 밑줄
  • del: 글 가운데 줄이 생김
  • div: 레이아웃 나누는 태그
  • span:스크립트를 범위를 위해 감싸주는 태그
  • hr: 줄 긇기
  • meta: 8가지 사이트에(크롬,마이크로 엣지,오피라 등)사이트에 창을 만드는데 사용함
  • p: 블록을 구성하는 태그
  • title: 마우스 포인트를 가져가면 말풍선이 나타남
  • a: 링크를 거는 태그
